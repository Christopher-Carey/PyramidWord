About

This web app is built with the MEAN stack.

  • MongoDB Database
  • Express Server
  • Angular For Front end
I utilize Angular components and Angular routing. I also implemented Bootstrap for styling.

When a word is entered into the input and submit is clicked the function runs first checking if the word has been checked before. I am using a database to store words that have been previously entered. If the word does not exists in the database it will run thought the algorithm to determine if it is a pyramid word. Then it will take that information and save it to the database for future reference.

This site is deployed on AWS with Windows Server using IIS.
